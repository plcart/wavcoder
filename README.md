# wavcoder
pipe function to record and encode audio from the browser 

## How to use

*Build output file to ./dist directory*

    npm run build

*Promise result at the end*

    const result = window.Wavcoder.pipe(stop, encoder, window.Wavcoder.blobToBase64)({});
